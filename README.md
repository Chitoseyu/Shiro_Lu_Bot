# Shiro_Lu_Bot

一個簡單的群組訊息刪除Bot

![boticon](https://i.imgur.com/yS3hLkE.png)

# 如何使用
 1. 將 bot 邀請到群組
 2. 給個有 **管理者權限** 的身份組
    
    若身份組權限不夠 
    <font color="red">刪除訊息功能 無法執行</font>
# 功能
- 批量刪除訊息 -> **有管理權限的使用者才能使用**
    
        clear 要刪除的行數  

    **刪除後無法復原,請小心使用**

- 人為操控 bot 發言
    
        sayd 要說的話
